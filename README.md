# blinky-board
This is a blinky board design I made using KiCad as my first hardware project. This board uses a 555 timer to flash LEDs. While working on this blinky board, I followed a tutorial, and I learned how to use the KiCad software. Figuring out where all the parts of the board went was very difficult at first, but I was able to get a hang of it in the end. The hardest part for me was importing a new footprint library, since a there were a lot of new terms to navigate. Overall, this project was super informative in learning the basics of PCB design.
